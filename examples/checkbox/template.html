Checkbox属性

属性名 类型 说明 可选值 默认值
disabled Boolean 是否禁用状态, 单向绑定 true | false false
checked Boolean 是否选中状态, 双向绑定 true | false false
halfChecked Boolean 是否半选状态, 双向绑定 true | false false
trueValue Object ng-true-value的实现 代替ture的某个值 true
falseValue Object ng-false-value的实现 代替false的某个值 false
